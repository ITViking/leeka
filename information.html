The

Leakage Detector

is an instrument that

is designed for the process industry and especially the Pharmaceutical industry where there is a high demand for process control and where the consequence of a leaking or damaged valve can give a high cost.

The Leakage Detector is typically connected to a process control system.

When sending information to the process control system, it can be registered in the control room if there is an alarm and information can be used to show history of the state of the valve.

History of events can also be retrieved directly at the Leakage Detector.

The Leakage Detector is typically place after a valve.

In this position it will detect if the valve is leaking.

The Leakage Detector has a basic light setting:

  • Green when no liquid is passing the Leakage Detector
  • Red light if liquid is passing
  • Yellow light if liquid is passing due to an actuated valve
  • An optional function is a light that is flashing if an unwanted event has passed

The light makes it easy for maintenance personal to detect if an incidence has occurred.
